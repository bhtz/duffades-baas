duffade BAAS
==========

parse-server backend as a service & dashboard for rapid prototyping


#### Requirements

* NPM
* Node >= 4.3
* MongoDB / PostgreSQL

#### Getting started

        npm install

        npm start

Go to your [dashboard](http://localhost:1337/dashboard/apps) !
